 - Manual de instalación -
A continuación, se presenta una guía de instalación del proyecto.
1. Primero ha de tener una distribución de Python 3.9.x instalada en su computadora, en caso de no tenerlo descárguelo en el siguiente enlace: https://www.python.org/downloads/ 
2. Una vez instalada la distribución se ha de abrir un terminal y ejecutar los siguientes comandos para la instalación de las librerías necesarias para ejecutar el proyecto.
	- pip install eel
	- pip install nltk
	- pip install tk
	- pip install speechrecognition
	- pip install ez_setup
	- pip install moviepy
3. Una vez instaladas estas librerías únicamente ha de ejecutar el archivo Audio.py en la carpeta del proyecto y seguir los pasos marcados en la guía de usuario.
